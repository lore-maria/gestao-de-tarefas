-- Gestão de Tarefas:

Este é um projeto simples de "Gestão de Tarefas" desenvolvido com: HTML, CSS e JavaScript . Ele permite adicionar, buscar, editar e excluir atividades de forma dinâmica.

Funcionalidades:

- Adicionar uma nova atividade com título, descrição e ID do usuário.
- Buscar atividades pelo ID do usuário.
- Editar o título de uma atividade.
- Excluir uma atividade.
- Listar todas as atividades disponíveis.

Estrutura do Projeto:

O projeto contém os seguintes arquivos:

- index.html → Estrutura da página.
- estilo.css → Estilização da interface.
- script.js → Funcionalidade dinâmica (CRUD de atividades).
- logo.png → Logotipo do projeto.

Tecnologias Utilizadas:

- HTML → Estruturação da página.
- CSS → Estilização e layout responsivo.
- JavaScript → Manipulação do DOM e integração com API.

Como Executar o Projeto:

1. Faça o download ou clone este repositório.
2. Abra o arquivo "index.html" em um navegador.
3. Utilize os formulários para adicionar, buscar e gerenciar atividades.

Obs.:

- O projeto utiliza a API pública https://jsonplaceholder.typicode.com/todos para simulação das requisições.
- O layout tem um design simples e responsivo, com navegação intuitiva.
